# 🗂️ 프로젝트 소개
**Waving(웨이빙)** 은 일상 속에서도 **소중한 지인과의 관계**를 이어갈 수 있도록 도와 주는 `지인 관리 앱`입니다. 
>어떤 말을 보낼지 고민하다가 연락을 미룬 적이 있으신가요?
일상 속에서도 소중한 지인과의 관계를 이어갈 수 있도록 빙빙이가 상황에 맞는 다양한 인사말을 추천해드려요!
